MCP Tool Development: Portfolio Management Server

Blog + Books + Podcasts Management

Challenge

Needed efficient way to manage dynamic content (blog posts, book recommendations, podcasts) on GitHub Pages portfolio. Required AI-assisted content creation with proper disclaimers and version control integration.

Solution & Architecture

Built comprehensive MCP (Model Context Protocol) server in Node.js with multiple tools:

  • Blog management: create, update, delete posts with AI disclaimer injection
  • Book recommendations: add, list, remove with Amazon integration
  • Podcast management: curated list with host and URL tracking
  • JSON metadata storage with automatic markdown generation
  • Integrated with Kiro CLI for seamless AI-assisted content creation

Business Impact

  • Reduced content publishing time from hours to minutes
  • Enabled AI-assisted blog writing with proper attribution
  • Created reusable MCP pattern for content management
  • Dynamic content loading improves site maintainability
Node.js MCP Protocol GitHub Pages AI Integration

HTTP-Based MCP Automation Server

70% Cost Reduction via Spot Instances

Challenge

Needed secure, scalable MCP server for automation workflows including webhook triggers, screenshot analysis, video creation, and AI-enhanced processing. Required enterprise-grade authentication and cost-effective deployment.

Solution & Architecture

Built HTTP-based MCP server with Java/Javalin featuring:

  • OAuth 2.0 + PKCE authentication for secure access
  • Deployed on ECS Fargate Spot reducing costs 70%
  • Terraform IaC with GitHub OIDC for keyless deployments
  • Webhook integration for event-driven automation

Business Impact

  • Reduced infrastructure costs by 70% using Spot instances
  • Eliminated static AWS credentials with OIDC
  • Enabled secure automation workflows
  • Scalable architecture handling variable workloads
