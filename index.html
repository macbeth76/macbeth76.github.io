HTTP-Based MCP Automation Server

70% Cost Reduction via Spot Instances

Challenge

Needed secure, scalable MCP server for automation workflows including webhook triggers, screenshot analysis, video creation, and AI-enhanced processing. Required enterprise-grade authentication and cost-effective deployment.

Solution & Architecture

Built HTTP-based MCP server with Java/Javalin featuring:

  • OAuth 2.0 + PKCE authentication for secure access
  • Deployed on ECS Fargate Spot reducing costs 70%
  • Terraform IaC with GitHub OIDC for keyless deployments
  • Webhook integration for event-driven automation

Business Impact

  • Reduced infrastructure costs by 70% using Spot instances
  • Eliminated static AWS credentials with OIDC
  • Enabled secure automation workflows
  • Scalable architecture handling variable workloads
